# Static / Motion / Reviewer Questions

## For Static Image Agent

- Can you preserve one exact dog across all panels: adult gray-and-white Siberian husky, symmetrical gray mask, narrow white blaze, left icy-blue eye, right amber-brown eye, red collar, small silver tag?
- If the tool cannot preserve identity across independent frames, can you generate one contact sheet instead of separate panels?
- Can the first approved Miso frame become the reference for all later frames?
- Can you leave text space clean enough for overlays to be added manually if generated text is garbled?

## For Motion / Video Agent

- Does the video tool support image reference, character reference, seed image, or first-frame conditioning for Miso?
- Can every shot reuse the same room geography: oak floor, off-white wall, white cabinet, stainless bowl, gray mat, blue chew toy?
- Can the first three seconds include a visible paw stomp or equivalent husky behavior proof?
- Can source audio or SFX preserve the paw stomp / husky-talking moment without making the dog seem distressed?
- If identity cannot be locked, should this remain a prompt pack rather than a generated-video deliverable?

## For Reviewer

- Does Miso remain the same dog in every shot?
- Are the mismatched eyes, red collar, silver tag, gray mask, and white blaze consistent?
- Does the room stay consistent across cuts?
- Does the overlay look playful rather than official legal, medical, or platform UI?
- Does the video avoid hard selling and keep the CTA comment-first?
- Does the first-three-second sequence communicate dinner delay -> complaint -> behavior proof?

## For Classifier / Modifier / Judge

- Classifier: should "breed-personality behavior proof" be tracked as a pet subtype or only a case-level note for now?
- Modifier: no durable knowledge-base mutation is recommended from this demo.
- Judge: if this concept becomes a real test, compare it against a broad cute control and another husky behavior-proof clip, not against unrelated pet product demos.

## Caption And Disclosure Rules

- Keep caption claims observable: "testing," "fits this drawer," "one wipe test," "bench check," "works in this trunk."
- Avoid unsupported superlatives: "best," "guaranteed," "safe for all," "vet approved," "factory certified," "fits every car."
- If a brand, affiliate link, free sample, paid placement, or TikTok Shop product is used, apply platform disclosure tools before publishing.
- Use comments for next-test selection only when the next test is real and safe.

## Shot Density

- Slots 1-4: 8-18 seconds unless comparison needs up to 25 seconds.
- Slots 5-8: 10-22 seconds, slower if animal welfare context is necessary.
- Slots 9-12: 12-25 seconds, with explicit fit/usage boundary on-screen.
- Minimum first-frame standard: viewer can identify the object, friction, or result before reading the caption.

## Updated Production Rule

A concept should not be approved because it has "a twist". It should be approved only when:

1. the twist is not generic;
2. the pet motive is credible;
3. the first 2 seconds prove the premise visually;
4. the payoff creates visible consequence;
5. a cold viewer can understand it without knowing the pet.
